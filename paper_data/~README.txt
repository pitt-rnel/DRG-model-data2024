README

This folder contains data associated with the paper: 

Farooqui J, Nanivadekar AC, Capogrosso M, Lempka SF, Fisher LE. The effects of neuron morphology and spatial distribution on the selectivity of dorsal root ganglion stimulation. J Neural Eng. 2024 Sep 4. doi: 10.1088/1741-2552/ad7760. PMID: 39231464.


This folder contains two subfolders:

cell_placements contains files with information about neuron placements in each of the three model types described in the paper

simulation_results contains files with neuron information and simulation outputs from each of the three model types in response to stimulation with each of the two electrodes, as described in the paper
